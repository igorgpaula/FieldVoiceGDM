Field Voice Tips

Português

  1. Este app funciona apenas online. Garanta que você tenha acesso à internet, por Wi-Fi ou dados móveis.
  2. Para falar uma nota, diga "Nota" antes para garantir o salvamento adequado de cada avaliação.
  3. Outras funcionalidades importantes que ajudam:
    "Próxima", "Pular" ou "Pula" - vai para a próxima avaliação/parcela.
    "Voltar" ou "Volta" - volta uma avaliação/parcela.
    "Parcela atual" - informa em qual parcela você está atualmente.
    "Parcela 2489" - pula para esta parcela; fale o número da parcela desejada.
    "Descrição" - ativa o modo de descrição a nível de parcela.
  4. Se você fechar o app, não se preocupe: quando abrir novamente, os dados estarão salvos.
  5. Qualquer dúvida ou sugestão, envie um email para ipaula@gdmseeds.com.

English

  1. This app works online only. Make sure you have internet access through Wi-Fi or mobile data.
  2. When speaking a score, say "Score" first to help ensure each trait score is saved correctly.
  3. Other helpful commands:
    "Next", "Skip" or "Advance" - moves to the next trait/plot.
    "Back" or "Previous" - goes back one trait/plot.
    "Current plot" - tells you which plot you are currently scoring.
    "Plot 2489" - jumps to that plot; say the plot number you want.
    "Description" - activates plot-level description mode.
  4. If you close the app, do not worry: when you open it again, your data will still be saved.
  5. Questions or suggestions: email ipaula@gdmseeds.com.

Español

  1. Esta app funciona solo en línea. Asegúrate de tener acceso a internet por Wi-Fi o datos móviles.
  2. Para dictar una calificación, di "Nota" antes para ayudar a guardar correctamente cada evaluación.
  3. Otros comandos útiles:
    "Siguiente", "Saltar" o "Avanzar" - va a la siguiente evaluación/parcela.
    "Volver" o "Anterior" - vuelve una evaluación/parcela.
    "Parcela actual" - indica en qué parcela estás actualmente.
    "Parcela 2489" - salta a esa parcela; di el número de parcela que deseas.
    "Descripción" - activa el modo de descripción a nivel de parcela.
  4. Si cierras la app, no te preocupes: cuando la abras de nuevo, los datos seguirán guardados.
  5. Preguntas o sugerencias: envía un email a ipaula@gdmseeds.com.
